## [1.0.0] - 2025-11-27

### Added

- **Portfolio Launch**: 15 architectural projects showcasing residential, commercial, and institutional work
- **Services Pages**: Home remodels, ADU design, and ROI dashboards sections
- **About Page**: Team bios and Uncia brand explanation
- **Contact Page**: Email and inquiry form for potential clients
- **Projects Archive**: Grid view of all 15 completed projects with filtering and search
- **Data Engineering Showcase**: ROI dashboard examples with cost estimation tools
- **Custom Logo & Favicon**: Uncia branding with SVG logo and favicon
- **GitHub Actions Workflow**: Custom Jekyll 4.3.4 build workflow for GitHub Pages deployment
- **Featured Projects Section**: Homepage showcase of 3 featured projects with direct links
- **Responsive Design**: Mobile-first design using Minimal Mistakes theme
- **Fast Build Performance**: Jekyll static site generation with ~0.5 second build time

### Technical

- Built with Jekyll 4.3.4 and Minimal Mistakes theme
- Ruby 3.2.2 environment configuration
